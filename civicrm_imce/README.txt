OVERVIEW
========
Integrates Drupal IMCE module with CiviCRM build in Editors CKeditor and TinyMCE. No configuartion is needed, just install the module.
The integration for TinyMCE is working with CiviCRM versions >= 3.3.2
For Versions below 3.3.2 you have to apply the patch file included in the module directory "tinymce.php.diff", for further information see http://issues.civicrm.org/jira/browse/CRM-7266

REQUIREMENTS
============
* CiviCRM
* IMCE
